=head1 PROPERTIES

=over

=item 'hadjustment' (Gtk2::Adjustment : default undef : readable / writable / construct / private)

The GtkAdjustment that determines the values of the horizontal position for this viewport

=item 'shadow-type' (Gtk2::ShadowType : default "in" : readable / writable / private)

Determines how the shadowed box around the viewport is drawn

=item 'vadjustment' (Gtk2::Adjustment : default undef : readable / writable / construct / private)

The GtkAdjustment that determines the values of the vertical position for this viewport

=back
