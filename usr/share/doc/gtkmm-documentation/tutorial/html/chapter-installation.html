Dependencies

Before attempting to install gtkmm 3.0, you might first need to install these other packages.

  • libsigc++ 2.0

  • GTK+ 3.0

  • cairomm

  • pangomm

  • atkmm

These dependencies have their own dependencies, including the following applications and libraries:

  • pkg-config

  • glib

  • ATK

  • Pango

  • cairo

  • gdk-pixbuf
