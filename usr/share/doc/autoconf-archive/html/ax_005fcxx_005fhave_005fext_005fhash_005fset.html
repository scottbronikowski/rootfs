ax_cxx_have_ext_hash_set

Synopsis

     AX_CXX_HAVE_EXT_HASH_SET

Description

Check if the compiler has ext/hash_set Eg: 

       #if defined(HAVE_EXT_HASH_SET)
       #include <ext/hash_set>
       #else
       #if defined(HAVE_STL)
       #include <hash_set>
       #else
       # Can't find hash_set header !
       #endif
       #endif
