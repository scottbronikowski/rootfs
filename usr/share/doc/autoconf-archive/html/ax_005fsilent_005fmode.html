ax_silent_mode

Synopsis

     AX_SILENT_MODE(on|off)

Description

Temporarily disable console output when running Autoconf macros. For example: 

       AX_SILENT_MODE(on)    dnl disable console output
       AC_PROG_CXX
       AX_SILENT_MODE(off)   dnl enable console output
       AC_PROG_RANLIB
