ax_gcc_install_dir

Synopsis

     AX_GCC_INSTALL_DIR(VARIABLE)

Description

AX_GCC_INSTALL_DIR(VARIABLE) defines VARIABLE as the gcc install directory. The install directory will be obtained using the gcc -print-search-dirs option. This macro requires AX_GCC_OPTION macro.

Thanks to Alessandro Massignan for his helpful hints.
