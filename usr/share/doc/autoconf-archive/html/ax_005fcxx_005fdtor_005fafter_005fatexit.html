ax_cxx_dtor_after_atexit

Synopsis

     AX_CXX_DTOR_AFTER_ATEXIT

Description

If the C++ compiler calls global destructors after atexit functions, define HAVE_DTOR_AFTER_ATEXIT. WARNING: If cross-compiling, the test cannot be performed, the default action is to define HAVE_DTOR_AFTER_ATEXIT.
