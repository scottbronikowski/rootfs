ax_check_awk_rshift

Synopsis

     AX_CHECK_AWK_RSHIFT([ACTION-IF-SUCCESS],[ACTION-IF-FAILURE])

Description

Check if AWK supports rshift() function. If successful execute ACTION-IF-SUCCESS otherwise ACTION-IF-FAILURE.

This work is heavily based upon testawk.sh script by Heiner Steven. You should find his script (and related works) at <http://www.shelldorado.com/articles/awkcompat.html>. Thanks to Alessandro Massignan for his suggestions and extensive nawk tests on FreeBSD.
