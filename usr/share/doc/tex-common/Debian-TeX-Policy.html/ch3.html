The Debian TeX sub-policy
Chapter 3 - TeX packages for the impatient

  • A package that only installs TeX input files, e.g. a new LaTeX package, should install them in the TEXMFMAIN tree (/usr/share/texmf/) at the place indicated by the TDS, see tds.html and File searching and libkpathsea / libkpse, Section 5.1, and register them in the maintainer scripts, usually by calling dh_installtex in debian/rules

  • Packages that add fonts, hyphenation patterns or formats, or want to change the basic configuration in texmf.cnf, need to follow the rules in Configuration update programs, Section 6.2 in addition to that.
