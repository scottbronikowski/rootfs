The Debian TeX sub-policy
Chapter 4 - Meta-packages and dependencies

The TeX Live collection of basic and add-on TeX packages provides some meta-packages for the convenience of users.

Depending on the texlive-* or tetex-* metapackages is only acceptable for editors, IDEs and other tools which handle user-generated code. TeX add-on packages, as well as automated input generators etc., must instead depend on a list of individual texlive packages which are actually used. [1]
