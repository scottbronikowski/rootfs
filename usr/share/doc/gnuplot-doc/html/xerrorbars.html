2.22 xerrorbars

The xerrorbars style is only relevant to 2D data plots. xerrorbars is like dots, except that a horizontal error bar is also drawn. At each point (x,y), a line is drawn from (xlow,y) to (xhigh,y) or from (x-xdelta,y) to (x+xdelta,y), depending on how many data columns are provided. A tic mark is placed at the ends of the error bar (unless bars is used—see bars for details). The basic style requires either 3 or 4 columns: 

          3 columns:  x  y  xdelta
          4 columns:  x  y  xlow  xhigh
