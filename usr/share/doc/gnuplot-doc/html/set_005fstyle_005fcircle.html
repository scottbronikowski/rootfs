3.24.61.7 set style circle

Syntax: 

           set style circle {radius {graph|screen} <R>}

This command sets the default radius used in plot style "with circles". It applies to data plots with only 2 columns of data (x,y) and to function plots. The default is "set style circle radius graph 0.02".
