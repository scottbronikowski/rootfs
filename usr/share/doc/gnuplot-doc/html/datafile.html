3.24.16 datafile

The datafile command options control interpretation of fields read from input data files by the `plot`, `splot`, and fit commands. Six such options are currently implemented.

  • set_datafile_fortran
  • set_datafile_nofpe_trap
  • set_datafile_missing
  • set_datafile_separator
  • set_datafile_commentschars
  • set_datafile_binary
