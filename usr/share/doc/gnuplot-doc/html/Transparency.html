1.4.2 Transparency

Gnuplot now supports several forms of transparency. Any object or plot element that uses a fill style can be assigned a transparency from fully opaque to fully transparent. Image or matrix data can be plotted with an alpha channel using the new plot style rgbalpha. See `fillstyle`, rgbalpha.
