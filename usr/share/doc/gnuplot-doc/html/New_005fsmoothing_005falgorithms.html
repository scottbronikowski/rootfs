1.4.7 New smoothing algorithms

New smoothing algorithms have been added for both 2- and 3-dimensional plots. `smooth kdensity` and `smooth cumul` can be used with `plot` to draw smooth histograms and cumulative distribution functions, resp. For use with `splot` several new smoothing kernels have been added to dgrid3d. See smooth dgrid3d.
