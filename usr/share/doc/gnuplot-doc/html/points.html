2.17 points

The `points` style displays a small symbol at each point. The command pointsize may be used to change the default size of the points. 1 or 2 columns of basic input data are required in 2D plots; 1 or 3 columns are required if 3D plots. See `style lines`. Additional input columns may be used to provide information such as variable point size or line color.
