SGML Entity Management

Abstract

This document provides a guidelines for implementation dependent Entity management of SGML entities for Debian systems. This defines the mapping of External Public Identifiers to System Identifiers. (In other words, this document covers the use of /usr/lib/sgml, entity naming, and catalog files.
