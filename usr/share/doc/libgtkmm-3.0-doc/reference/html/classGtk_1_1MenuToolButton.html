Detailed Description

A Gtk::ToolItem containing a toggle button.

A MenuToolButton is a Gtk::ToolItem that contains a menu.

Glib::SignalProxy0< void > Gtk::MenuToolButton::signal_show_menu ( )
Slot Prototype:
void on_my_show_menu()

The signal_show_menu() signal is emitted before the menu is shown.

It can be used to populate the menu on demand, using Gtk::MenuToolButton::get_menu().

Note that even if you populate the menu dynamically in this way, you must set an empty menu on the Gtk::MenuToolButton beforehand, since the arrow is made insensitive if the menu is not set.
