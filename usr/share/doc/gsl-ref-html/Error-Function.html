7.15.1 Error Function

— Function: double gsl_sf_erf (double x)
— Function: int gsl_sf_erf_e (double x, gsl_sf_result * result)

These routines compute the error function erf(x), where erf(x) = (2/\sqrt(\pi)) \int_0^x dt \exp(-t^2).
