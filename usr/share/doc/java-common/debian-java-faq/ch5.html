5.1 What Java compilers are available in Debian?

  • openjdk-6-jdk

  • sun-java6-jdk (non-free)

  • gcj. Compiles Java source to native code, also source to bytecode, or bytecode to native code.

  • The deprecated jikes in Lenny.
