sect1info — Meta-information for a Sect1

Synopsis

Content Model

Attributes

Common attributes

Description

Like the other “info” elements, Sect1Info contains meta-information about the section of the document in which it occurs.

Processing expectations

Suppressed. Many of the elements in this wrapper may be used in presentation, but they are not generally printed as part of the formatting of the wrapper. The wrapper merely serves to identify where they occur.

Future Changes

AuthorBlurb and Affiliation will be removed from the inline content of Sect1Info in DocBook V4.0. A new wrapper element will be created to associate this information with authors, editors, and other contributors.

Parents

These elements contain sect1info: sect1.

Children

The following elements occur in sect1info: abbrev, abstract, address, affiliation, artpagenums, author, authorblurb, authorgroup, authorinitials, bibliocoverage, biblioid, bibliomisc, bibliorelation, biblioset, bibliosource, citebiblioid, citetitle, collab, confgroup, contractnum, contractsponsor, contrib, copyright, corpauthor, corpcredit, corpname, date, edition, editor, firstname, graphic, honorific, indexterm, invpartnumber, isbn, issn, issuenum, itermset, keywordset, legalnotice, lineage, mediaobject, modespec, orgname, othercredit, othername, pagenums, personname, printhistory, productname, productnumber, pubdate, publisher, publishername, pubsnumber, releaseinfo, revhistory, seriesvolnums, subjectset, subtitle, surname, title, titleabbrev, volumenum.
