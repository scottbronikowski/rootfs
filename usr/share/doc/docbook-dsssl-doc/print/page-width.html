%page-width%

Name

%page-width% -- Specifies the page width

Synopsis

%page-width%

Description

Identifies the width of the page (length in the writing direction). It is usually controlled by the %paper-type% parameter.

Author

Norman Walsh, <ndw@nwalsh.com>

Source Code

(define %page-width%
  ;; Specifies the page width
  (case %paper-type%
    (("A4landscape") 297mm)
    (("USletter") 8.5in)
    (("USlandscape") 11in)
    (("4A0") 1682mm)
    (("2A0") 1189mm)
    (("A0") 841mm)
    (("A1") 594mm)
    (("A2") 420mm)
    (("A3") 297mm)
    (("A4") 210mm)
    (("A5") 148mm)
    (("A6") 105mm)
    (("A7") 74mm)
    (("A8") 52mm)
    (("A9") 37mm)
    (("A10") 26mm)
    (("B0") 1000mm)
    (("B1") 707mm)
    (("B2") 500mm)
    (("B3") 353mm)
    (("B4") 250mm)
    (("B5") 176mm)
    (("B6") 125mm)
    (("B7") 88mm)
    (("B8") 62mm)
    (("B9") 44mm)
    (("B10") 31mm)
    (("C0") 917mm)
    (("C1") 648mm)
    (("C2") 458mm)
    (("C3") 324mm)
    (("C4") 229mm)
    (("C5") 162mm)
    (("C6") 114mm)
    (("C7") 81mm)
    (("C8") 57mm)
    (("C9") 40mm)
    (("C10") 28mm)))
