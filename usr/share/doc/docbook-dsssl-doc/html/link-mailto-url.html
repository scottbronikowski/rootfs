Name

%link-mailto-url% -- Mailto URL for LINK REL=made

Synopsis

%link-mailto-url%

Description

If not #f, the %link-mailto-url% address will be used in a LINK REL=made element in the HTML HEAD.

  ;; Mailto URL for LINK REL=made
